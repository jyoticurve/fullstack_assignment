Aspects HTML HTML5
Document Structure Uses various div and spans for layout styling. Introduces new semantic elements like header, nav, footer, article, section for a structured document.
Multimedia Requires third-party plugins like Adobe Flash for multimedia content. Provides native support for multimedia through audio and video element.
Forms Basic form elements available, limited input types. Adds new input types (input type="email", input type="date" etc.) and attributes for improved form handling.
Responsive Web Design Requires additional CSS and media queries for responsive design. Promotes responsive web design through media queries and fluid layouts.
Drag and Drop Limited native support for drag-and-drop interactions. Introduces native support for drag-and-drop operations.
Offline Applications Lacks native support for creating offline web applications. Introduces the Application Cache(AppCache) for creating offline-capable web apps.
Web Workers No support for multi-threaded javascript. Introduces web workers for running scripts in the background without freezing the main UI.
