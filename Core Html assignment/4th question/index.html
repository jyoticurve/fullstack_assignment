Ordered List of HTML tags:

  1. !DOCTYPE

    It defines the document type and version of html used in a webpage.

  2. html

    The root element of an HTML document, containing all other elements.

  3. head

    It contains metadata about the HTML document such as title, character and linked stylesheets.

  4. title

    It sets the title of the webpage, which appears in the browser's title bar or tab.

  5. meta

    Provides metadata about the document, such as character encoding and viewport settings.

  6. style

    Contains CSS code for styling the webpage.

  7. li

    It represents a list item within ul(unordered list) and ol(ordered list).

  8. img

    It embeds image in the webpage.

  9. table

    It defines a table structure for displaying data in rows and columns.

  10. form

    It creates a form for user input, which can include text fields, checkboxes, and buttons.
