Description List of full stack web development tech stack:
HTML (Hypertext Markup Language):
It is the standard markup language for creating web pages and it also describes the structure of a web page.

CSS (Cascading Style Sheets):
It is the language we use to style an HTML document. It describes how HTML elements should be displayed.

Javascript:
It is a programming language that is one of the core technology of the World Wide Web alongside HTML and CSS. It adds functionality to your structure or web page.

Servers:
A server is a computer program that can be accessed remotely over a network to provide resources and services to another computer program.

Web Frameworks:
It is a software framework that is designed to support the development of web applications including web services, web resources and web APIs like angular, ruby on rails.

Database:
It is information that is set up for easy access, management and updating. it is used for storing, maintaining and accessing any sort of data.

SQL(Structured Query Language):
It helps them to describe the structured data. It is used to perform operations on the records stored in the database, such as updating, inserting, deleting and modifying database tables, views etc.
